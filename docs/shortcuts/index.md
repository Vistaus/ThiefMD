# Changing the UI

* `Ctrl + ,` - Access preferences window

## Full Screen

* `F11` - Toggle Full Screen

## Editor Views

* `Ctrl + 1` - Editor Only
* `Ctrl + 2` - Editor and Sheets View
* `Ctrl + 3` - Editor, Sheets, and Library View

# Sheets Management

* `Ctrl + S` - Force Save
* `Ctrl + N` - New Sheet
* `Ctrl + Shift + P` - Launch Preview
